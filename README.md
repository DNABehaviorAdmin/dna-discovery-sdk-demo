# DNA Behavior Discovery SDK - Integration Demo

Welcome to the **DNA Behavior Discovery SDK integration demo**. 

This repository provides a hands-on, interactive example of how to embed the DNA Behavior Discovery assessment into a modern web application using React, Vite, and Tailwind CSS.

## Getting Started

1. **Install Dependencies:**
   ```bash
   npm install
   ```
2. **Run the Development Server:**
   ```bash
   npm run dev
   ```
3. **Configure the Demo:**
   Launch the app in your browser and follow the on-screen Setup wizard to dynamically inject your `Account ID`, `Self-Registration ID`, and cryptographic keys into the session.
